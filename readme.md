# Reading Log

A running intellectual log of books, papers, and gyan I read/acquire — [view site](https://varun-k-r.github.io/TLDR/)
## What this is

This is a public record of how I think through material as I read — not a literature review, not a summary service. Entries contain tentative hypotheses, disagreements with the literature, cross-references to other thinkers, and positions held with explicit uncertainty. Entries may be wrong and are meant to be. Readers are assumed to be academically literate.

## What this is not

This is not a polished research output. It is discursive, first-person, and candid. The point is to maintain a log in public, not to perform expertise.

## Structure

Each entry follows a consistent format:

- **Summary** — 2–4 sentences distilling the core argument
- **Notes & commentary** — observations on the text, lightly structured
- **My take** — interpretation, disagreements, and tentative conclusions
- **Further reading** — sources worth following up, checked against USC Libraries

Figures are stored under `\figs\<book-slug>\` and embedded inline at the point of reference.
